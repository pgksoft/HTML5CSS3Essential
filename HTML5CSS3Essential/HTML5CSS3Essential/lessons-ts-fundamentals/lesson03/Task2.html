Lesson 03: Generics.

Task 2.

Создать словарь собственных определений, используя Generic function. Внутри должно быть определение для 3 свойств – ключ, значение, описание (различных типов данных). Для получения или записи использовать get/set реализацию доступа. Также для полей нужно использовать модификаторы доступа (на Ваш выбор). В итоге должен получится словарь терминов, принимающий на входящий параметр различные типы данных для реализации.
