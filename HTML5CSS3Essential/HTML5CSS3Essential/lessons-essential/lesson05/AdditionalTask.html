Additional task

Создайте страницу с несколькими элементами div, p, a. span. Нескольким элементам кстановите стиль с помощью CSS правил. Нескольким элементам установите уникальный ID. Напишите сценарий, который с помощью document.querySelectorAll и document.querySelector будет выбирать со страницы элементы:
  • по имени тэга и устанавливать им зеленую рамку
  • по классу и устанавливать им красную рамку
  • по id и устанавливать им рамку синего цвета
