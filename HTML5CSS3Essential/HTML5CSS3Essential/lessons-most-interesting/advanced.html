Особенностью решения данной задачи стало использование aspx-страницы. В Visual Studio 2017, в списке возможных элементов для добавления в WEB-проекты, он значится как Web Form. Мне, если так можно сказать, надоело включать много текстовой информации в HTML, JS и TS файлы. Общая информация об отелях берется из текстовых файлов на сервере.

При сборе предметной информации понял, что будет скучно отображать для каждого топового отеля только одно изображение. Возникла идея заменить одно изображение слайдером. Т.е. на странице, в данном примере, должно быть четыре слайдера. Поэтому решил усовершенствовать свой слайдер, который был создан при решении задачи из курса TypeScript Fundamentals.
